Cara Penggunaan

1. Laporan Harian : Untuk menampilkan hasil Inputan Data Absensi pada hari ini.

2. Laporan Bulanan : Untuk menampilkan hasil rekapan siswa setiap harinya.

3. Input Kehadiran : Untuk mendata siswa dengan keterangan yang sudah tersedia.


Note : Ketika anda menekan tombol INPUT KEHADIRAN itu artinya anda sudah mengAbsensi seluruh siswa tanpa harus mengklik tombol Hadir satu per satu.Jadi, anda tidak perlu repot-repot mengklik tombol HADIR.

Nah, jika ada siswa yang tidak berangkat anda tinggal mengklik tombol keterangan yang sudah disediakan dan fungsinya adalah untuk mengUpdate data tersebut.
